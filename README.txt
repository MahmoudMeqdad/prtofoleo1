منصة مساعد المساقات الجامعية — Deep Source Mode v2

هذه النسخة مبنية على النسخة البحثية، وتضيف استرجاعًا عميقًا متعدد المراحل.

الميزات الجديدة
================

1. وضعان للاسترجاع:
   /deep
   تفعيل Deep Source Mode.

   /precise
   العودة إلى البحث السريع المركز.

   /mode
   عرض الوضع الحالي والمصدر المحدد.

2. إعادة صياغة متعددة للاستعلام:
   ينشئ النظام عدة صيغ بحث تكمل بعضها بدل الاعتماد على صيغة السؤال فقط.

3. بحث هجين:
   - بحث دلالي باستخدام Gemini Embeddings وChroma.
   - بحث لفظي بالكلمات والمفاهيم داخل المقاطع المخزنة.
   - دمج النتائج وإزالة التكرار.

4. توسيع السياق:
   عند استرجاع مقطع من صفحة معينة، يجلب النظام المقاطع من الصفحات المجاورة
   ضمن DEEP_NEIGHBOR_RADIUS.

5. فحص كفاية الأدلة:
   قبل الإجابة، يقرر النظام هل الأدلة تكفي أم لا، ويسجل:
   - sufficient
   - confidence
   - missing_evidence

6. جولة بحث ثانية:
   إذا كانت الأدلة غير كافية، يولد استعلامات متابعة وينفذ بحثًا إضافيًا تلقائيًا.

7. إجابة موثقة:
   يطلب من Gemini الاستشهاد بالأدلة داخل النص بعلامات مثل [S1] و[S2].
   يمنع التخمين ويصرح عندما يكون الدليل ناقصًا.

8. قياسات بحثية جديدة:
   - عدد استعلامات Deep Mode.
   - متوسط الثقة.
   - نسبة كفاية الأدلة.
   - عدد مراحل الاسترجاع.
   - عدد المرشحين والأدلة النهائية.
   - تصديرها عبر /export_logs.

الإعدادات
=========

أضف إلى .env:

DEEP_MODE_DEFAULT=false
DEEP_SEMANTIC_K=12
DEEP_FINAL_K=18
DEEP_QUERY_VARIANTS=4
DEEP_LEXICAL_K=12
DEEP_NEIGHBOR_RADIUS=1
DEEP_MAX_CONTEXT_CHARS=80000
DEEP_EVIDENCE_THRESHOLD=0.65

التشغيل على Windows
===================

1. افتح CMD داخل مجلد المشروع.
2. فعّل البيئة:

.venv\Scripts\activate

3. حدّث المتطلبات:

pip install -U -r requirements.txt

4. تأكد من وجود ملف .env والمفاتيح الصحيحة.
5. شغّل:

python bot.py

الاستخدام
=========

/deep
ثم اطرح سؤالًا يحتاج ربط أكثر من صفحة أو فصل.

/precise
للأسئلة المباشرة والسريعة.

/select 1
لتقييد البحث بملف واحد.

/all
للبحث في جميع المصادر.

/research_stats
لعرض مؤشرات التجربة.

/export_logs
لتصدير السجلات البحثية.

ملاحظات
=======

- Deep Source Mode يستهلك طلبات وتوكنات أكثر من Precise Mode.
- لا يحتاج الانتقال من النسخة السابقة إلى حذف vector_db لأن نموذج Embeddings لم يتغير.
- تم إصلاح اختيار المصدر بحيث يبقى المصدر المحدد نشطًا بعد /select.
